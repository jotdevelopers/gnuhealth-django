About Life

Life is a hospital information management system with data visualization feature. Life is a web based system developed using DJANGO Framework which is a python web developent framework. Progres SQL is used as database for the project.

Departments

There are many hospital departments, staffed by a wide variety of health professionals, with some crossover between departments.

Dentists

This department deals with patients who are having dental problems.

Neurology

This department deals with disorders of nervous system, including brain and spinal cord.

Opthalmology

Eye department provides services including laser treatments, optometry (sight testing).

Orthopedics

This department treat problems that affect musculoskeletal system as muscles, joints etc.

Cancer Department

It provides radiotherapy and chemotherapy treatments for cancerous tumours.

ENT Department

It provides care for variety of problems, including: general ear, nose and throat diseases
